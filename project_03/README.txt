Давайте создадим простую программу на Python, которая будет использовать API MangAPI3 с сайта RapidAPI для получения информации о манге. 

▎Шаги для выполнения:

1. Зарегистрируйтесь на RapidAPI: Вам нужно создать учетную запись на RapidAPI и подписаться на API MangAPI3, чтобы получить ваш API ключ.

2. Установите необходимые библиотеки: Убедитесь, что у вас установлены библиотеки requests. Вы можете установить её с помощью pip:

   pip install requests
